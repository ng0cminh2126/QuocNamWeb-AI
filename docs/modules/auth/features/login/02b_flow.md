## ğŸš¦ Route Protection

### Public Routes

- `/login` - Login page
  - Redirect to `/portal` if already logged in

### Protected Routes

- `/` - Redirect to `/portal` if logged in, else `/login`
- `/portal` - Main portal (requires auth)
- `/portal/*` - All portal sub-routes (requires auth)
