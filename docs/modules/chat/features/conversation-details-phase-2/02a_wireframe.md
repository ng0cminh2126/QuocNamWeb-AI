## â™¿ Accessibility

### Image Message

- `role="button"` - Clickable image
- `aria-label="View full-size image"` - Screen reader label
- `tabindex="0"` - Keyboard focusable
- `onKeyPress` - Enter/Space to open preview

### Preview Modal

- `role="dialog"` - Modal dialog
- `aria-modal="true"` - Lock focus within modal
- `aria-labelledby` - Modal title
- Focus trap - Tab cycles within modal
- Focus return - Return to trigger element on close

### Loading States

- `aria-busy="true"` - Loading indicator
- `aria-live="polite"` - Announce loading completion

### Error States

- `role="alert"` - Error announcement
- `aria-live="assertive"` - Immediate announcement
