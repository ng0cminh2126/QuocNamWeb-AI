# [BƯỚC 1] Requirements - Pinned & Starred Message

## Functional Requirements
- Users can pin messages in a chat group
- Users can star messages for personal reference
- Pinned messages are visible to all group members
- Starred messages are private to the user
- UI indicators for pinned/starred status

## UI Requirements
- Pin/star icons on each message
- Pinned messages section at top of chat
- Starred messages accessible via filter/tab

## Security Requirements
- Only group members can pin messages
- Only message owner can star/unstar
